# ✅ Dream OS - Accessibility Audit (WCAG 2.2 AA)

## 🔍 PERCEIVABLE
- [ ] Icon punya aria-label / .sr-only
- [ ] Contrast text ≥ 4.5:1 (normal), ≥ 3:1 (large)
- [ ] Zoom 200% → layout tidak broken

## ⌨️ OPERABLE
- [ ] Touch target ≥ 44x44 px
- [ ] Navigasi keyboard lengkap (Tab/Enter/Space)
- [ ] Focus indicator terlihat jelas

## 🧠 UNDERSTANDABLE
- [ ] Bahasa konsisten (ID/EN)
- [ ] Error message jelas & actionable

## 🛡️ ROBUST
- [ ] HTML/ARIA valid
- [ ] Compatible dengan TalkBack / VoiceOver

## 📋 Testing Manual
1. Aktifkan TalkBack (Android Settings > Accessibility)
2. Navigasi hanya pakai keyboard
3. Zoom 200%, cek layout
