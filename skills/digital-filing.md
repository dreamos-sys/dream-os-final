# SKILL: Digital Filing System (Dream OS)
- **Modularity**: Satu dokumen = Satu file Markdown. Jangan ditumpuk.
- **Indexing**: Gunakan format [YYYY-MM-DD]-[KODE]-[JUDUL].md.
- **Searchability**: Setiap file harus punya Metadata/Frontmatter di bagian atas.
- **Performance**: Gunakan lazy-loading jika menampilkan daftar dokumen yang banyak (>50).
