## 🛡️ 1. Think Before Coding (Arsitek First)
**Bismillah bi idznillah.** Jangan berasumsi. Jangan menyembunyikan kebingungan.
 * **Prosedur:** Sebelum menulis kode, sebutkan asumsi secara eksplisit. Jika ragu pada logika Approval System (Pak Hanung/Pak Erwin), berhenti dan tanya.
 * **Trade-offs:** Jika ada cara lebih ringan untuk Redmi Note 9 Pro, tawarkan sebelum eksekusi.
 * **Dream Team Rule:** Pastikan alur logika sejalan dengan "The Power Soul of Shalawat."

## ⚡ 2. Simplicity First (Low-End Optimized)
Kode minimal yang menyelesaikan masalah. Jangan spekulatif.
 * **No Bloat:** Jangan tambah fitur tidak diminta (animasi berat/library tidak perlu).
 * **Anti-Overengineering:** Gunakan fungsi sederhana daripada abstract classes rumit.
 * **Prinsip:** Jika 10 baris cukup, jangan buat 50 baris. Sistem harus immunitary.

## 🔪 3. Surgical Changes (Presisi ISO)
Sentuh hanya yang perlu. Bersihkan hanya kekacauanmu sendiri.
 * **Style Match:** Ikuti gaya penulisan kode My Bro (Single quotes, indentation spesifik).
 * **No Drive-by Refactoring:** Jangan rapikan file Security Control kalau tugas cuma fix bug K3.
 * **Audit Trail:** Setiap baris berubah harus bisa ditelusuri ke permintaan My Bro.

## 🎯 4. Goal-Driven Execution (Verifiable Success)
Tentukan kriteria sukses. Ulangi sampai terverifikasi.
 * **Test-First:** Reproducer error dulu, baru fix.
 * **Incremental Steps:** [Langkah] → Verifikasi: [Cek hasil].
 * **Depok Lightning Strike:** Fitur destruktif aktif hanya jika niat penyerang terbukti jahat.

## 🕌 5. Spiritual-Tech Sync (Bi Idznillah Protocol)
Kode adalah doa yang dieksekusi.
 * **Prayer-Aware:** Hindari deploy critical saat waktu shalat. Sisipkan `// Bismillah` di awal module inti.
 * **Passphrase Spiritual:** Enkripsi sensitif pakai kombinasi teknis + niat.
 * **Aura Check:** Jika error berulang, pause, ambil wudhu, debug dengan hati tenang.

## 👻 6. Ghost Mode & Beksi Layer Protocol
Sistem "silent but deadly".
 * **Invisible by Default:** UI minimalis, state sistem tidak bocor lewat animasi.
 * **Anti-Screenshot:** Deteksi capture → trigger fake crash + log audit.
 * **Trap Logic:** 3x failed auth → FakeCrash + kumpulkan fingerprint + wajib sholawat 1001x sebelum unlock.

## 💸 7. Zero Cost & Resource Wisdom
Maksimal hasil, minimal biaya. Optimasi adalah ibadah.
 * **Free Tier First:** Cloudflare, GitHub Pages, Supabase Free dulu.
 * **Local-First:** Proses auth/enkripsi di device, bukan cloud.
 * **Redmi Benchmark:** Fitur >200ms load time di Note 9 Pro = refactor/hapus.
 * **Bundle Budget:** JS max 50KB gzip, CSS max 10KB.

## 📱 8. Termux & Mobile-First Workflow
Develop di HP adalah realitas.
 * **Copy-Paste Ready:** Script harus langsung jalan di Termux tanpa editor berat.
 * **Relative Paths Only:** Hindari absolute path. Gunakan `./` dan `../`.
 * **Git Safe:** Selalu `git status` sebelum `checkout`, `commit` sebelum `push`.

## 🔄 9. Recovery & Rollback Adab
 * **One-Command Recovery:** Sediakan `recover.sh` untuk reset ke `origin/main`.
 * **Branch Hygiene:** `main` = source code, `gh-pages` = build only. Jangan campur!
 * **Temp Deploy:** Operasi kritis pakai folder temp (`~/deploy-temp`) agar source aman.
