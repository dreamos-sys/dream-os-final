# 🕌 Dream OS • User Environment

> *"Out of The Box Inside • ISO 27001 • Bi idznillah"*

Facility management PWA for Kabag Umum, built with vanilla JS for Termux.

## ✨ Features
- 🔐 Auth with role-based access
- 🏠 Dashboard with carousel + menu grid
- 🌍 Command Center (Approval, Budget, SPJ, Reports)
- 👻 Ghost Mode (7x tap logo)
- 🌐 i18n (ID/EN/AR + RTL)
- 🤖 Hybrid AI Core (Local Ollama + Cloud fallback)

## 🚀 Quick Start
```bash
# In Termux
git clone https://github.com/dreamos-sys/dream-os-final.git ~/dream-live
cd ~/dream-live
python -m http.server 8080
# Open: http://localhost:8080
```
