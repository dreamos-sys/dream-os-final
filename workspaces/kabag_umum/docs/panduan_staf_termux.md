# 📖 PANDUAN CEPAT: COMMAND CENTER STAF LAPANGAN

Selamat datang di ekosistem **Dream OS**. Gunakan panduan ini untuk mengelola tugas harian tanpa ribet kertas!

## 1. Cara Masuk ke Ruang Kerja
Ketik perintah ini di terminal:
* `cc-koor` (Masuk ke Ruang Kerja Koordinator Umum)

## 2. Cara Cek Tugas Harian
Setelah masuk, ketik:
* `m-refresh`
Lalu buka file `workspace_index.md` untuk melihat apa yang harus dikerjakan hari ini.

## 3. Cara Booking Ruangan/Alat (Anti-Double)
Gunakan perintah `m-book` dengan format:
`m-book "[Nama_Alat]" "[Tanggal]" "[Mulai]" "[Selesai]" "[Nama_Sultan]"`

*Contoh:*
`m-book "Aula" "2026-04-25" "08:00" "10:00" "Ahmad_Lapangan"`

## 4. Aturan Emas (M-Standards)
- **Cek dulu baru jalan**: Selalu `ls` folder `tasks/` untuk lihat update.
- **Lapor AI**: Kalau bingung, tanya asisten dengan ketik `m-koor "Saya bingung poin X, tolong jelasin"`.
